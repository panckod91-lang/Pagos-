# Vale USD

Mini app HTML/PWA para registrar pagos diarios en pesos y cierres semanales en USD.

## Archivos

- `index.html`
- `manifest.json`
- `sw.js`
- `icon.svg`

## Apps Script usado

## Uso

1. Subir estos archivos a GitHub.
2. Publicar con Cloudflare Pages.
3. Abrir la app.
4. Entrar como:
   - Tincho / PIN 1234
   - Vale / PIN 0000

Los PIN salen de la hoja `config`.
